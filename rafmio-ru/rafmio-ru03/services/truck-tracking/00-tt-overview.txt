Приложение для информации по регистрации перевозвок.

Базовый функционал:
1.Хранить в БД данные по перевозкам. Поля БД уже определены файлом "Данные по
  перевозкам"
2.Добавлять данные по перевозкам
3.Фильтровать и выводить нужные данные по перевозкам
4.СУБД - предварительно решено использовать PostgreSQL


- уметь добавлять запись html
- уметь добавлять запись в СУБД через формы
- уметь фильтровать данные
- на первое время выводить данные из *.go файла, потом *.txt, потом *.json файлов
- научиться добавлять запись в файл
